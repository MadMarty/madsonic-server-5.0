Installation

To install Madsonic, double-click the Madsonic Installer and follow the on-screen instructions. The Installer will tell you when the installation process is complete. Madsonic is installed as a Windows service, and is started automatically when the computer starts. Double-click the Madsonic icon in the Windows System Tray to launch the Madsonic control panel.

By default, Madsonic is accessible on http://localhost:4040/, but the port number and URL can be changed from the control panel.

If you are upgrading an existing installation of Madsonic, your previous settings (stored in c:\subsonic) will be preserved.
